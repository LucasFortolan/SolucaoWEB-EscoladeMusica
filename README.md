# Solução Web para Gestão Financeira e Operacional de uma Escola de Música

Este projeto está em desenvolvimento e tem como objetivo criar um sistema web para auxiliar a gestão administrativa da instituição musical **Passo a Passo**, localizada em Araras-SP.

A aplicação busca substituir processos manuais realizados atualmente em planilhas, oferecendo uma solução digital para organização de alunos, professores, pagamentos, comprovantes e relatórios administrativos.

## Documentação

A parte escrita do trabalho está sendo desenvolvida em um documento separado, contendo introdução, caracterização da instituição, problemática, escopo, diagrama UML, tecnologias utilizadas, resultados esperados, cronograma e referências bibliográficas.

## Funcionalidades previstas

- Cadastro de alunos
- Cadastro de professores
- Registro de pagamentos
- Controle de mensalidades pagas e pendentes
- Geração de comprovantes em PDF
- Relatórios de inadimplentes
- Relatórios de mensalidades pagas
- Relatórios de alunos por professor
- Dashboard com indicadores administrativos
- Configurações da instituição
